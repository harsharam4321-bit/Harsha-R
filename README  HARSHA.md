## Overview
The Expense Tracker System is a menu-driven, console-based Python application that helps users record and manage daily expenses efficiently.

## Problem Addressed
Manual expense tracking can cause data loss, calculation errors, and poor budgeting decisions. This project provides a simple digital solution.

## Features
- Add expense details
- View all expenses
- Search expense by ID
- Delete expense records
- File handling for permanent storage

## Python Concepts Used
- Lists and Dictionaries
- Functions
- Conditional Statements
- Loops
- Exception Handling
- File Handling

## How to Run
1. Open Expense_Tracker_Assignment.ipynb in Jupyter Notebook
2. Run all cells
3. Use the menu to manage expenses

## Input Guidelines
- Expense ID must be numeric
- Amount should be a number
- Use menu numbers only

## File Details
- Data File: expenses.txt
- Automatically created if not available
